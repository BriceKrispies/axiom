Axiom — Runtime Asset Streaming

The engine boots fast — this page is interactive before any asset finishes — then streams assets in the background, async and in parallel, driven by the deterministic scheduler in the axiom-assets module (priority + dependency order within a concurrency budget). The app performs the fetches the scheduler asks for and feeds the outcomes back each frame. Fetched bytes are opaque blobs here; decoding is out of scope.
